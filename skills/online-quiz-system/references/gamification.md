# Gamification / Motivation Layer

"Answer questions, see a score" gets boring within a week. Borrow the engagement loop from Duolingo / casual
games and layer it on the data you already collect. The golden rule: **tie every reward to *quality* learning**
(hard questions, wrong-answer review, weak topics) so learners don't just farm easy questions to game the system.

## Core loop
```
open app → see daily quests → practice (challenge / review)
  → gain XP + coins + streak → level up / finish quests / open chest
  → spend coins on cosmetics & items → want to beat a friend → come back tomorrow
```

## Phase A — personal growth (cheap, high impact; build this first)
- **XP + player level** (separate from subject levels). Award XP per question, with multipliers for difficulty,
  wrong-answer review (×1.5), and weak topics (×1.3). Animated level bar somewhere prominent.
- **Daily quests:** 3 small refreshing tasks ("answer 15", "review 5 wrong-answers", "do one math round").
  Completion grants XP + coins. Cap them — completion, not endless grinding, is the goal.
- **Streak with a freeze item** ("makeup card") so one missed day doesn't crush a long streak. Milestone rewards.
- **Achievement badges:** "7-day streak", "topic mastered", "100 correct", "50 wrong-answers conquered".

## Phase B — personalization (medium effort)
- **Shop:** spend coins on avatar skins, theme colors, frames, and items (hint card, double-XP card, freeze).
- **Study companion / pet:** a creature that grows as you practice and can be customized — gives kids something
  to nurture and show off.
- **Public profile:** level, badges, pet, best records — shareable to friends.

## Phase C — social & competitive (high effort, high stickiness)
- **Friends:** add classmates/siblings, see progress, cheer each other.
- **1v1 duel (PK):** same paper, compete on speed + accuracy (async is fine). Duel rating.
- **Weekly XP leaderboard** among friends/class, reset weekly (extend the contest leaderboard you already have).
- **Seasons + cosmetic gacha:** monthly season points → limited skins; transparent odds, cosmetics only — never
  pay-to-win on actual learning.
- **Boss battle:** a weekly themed hard-question set; beating it grants a big reward (solo or team).

## Health / anti-addiction
- Daily quests have a ceiling; don't reward infinite grinding.
- Rewards are steady companionship, not slot-machine variable reward. Gacha is cosmetic-only with shown odds.
- Give parents a view of learning-time vs gamification-interaction balance.

## Suggested MVP
Just Phase A's four items (XP+level bar, 3 daily quests, coins + a tiny cosmetic shop, badge wall) already lifts
daily retention noticeably and reuses existing attempt data. Add養成/social later, once learners actually return
daily.
