## 2. 檔案內部格式(已抽樣驗證)

每題自帶結構化標記,**可以自動解析**:

```
題號：0810001 難易度：易 學習內容：N-7-3 知識點：JMA010101010000
（  ）題目內容…(A)選項1　(B)選項2　(C)選項3　(D)選項4
《答案》D
詳解：…
```

### 對應到系統題庫欄位(自動取得,不需人工標)

| 系統欄位 | 來源 |
|---------|------|
| `id` | 題號 |
| `subject` / `topic` | 資料夾與檔名(如:數學/第1冊/1-1負數與數線) |
| `difficulty` | 難易度:易=1~2、中=3、難=4~5(會考真題依年度另行加權) |
| `question` `options` `answer` `explanation` | 內文解析 |
| `tags` | 學習內容課綱代碼(N-7-3)、知識點代碼、來源分類(精選題/習作/素養題/會考真題) |

## 3. 轉換流程

```
.doc(2240個)
  → Word COM 批次開啟,抽出文字 + 匯出內嵌圖片/數學式
  → 解析腳本(regex 切題)→ data/questions/*.json
  → 產出「轉換報告」:成功題數、含圖題數、解析失敗清單
  → 你抽查確認 → 匯入 Supabase
```

### 已知風險與對策

| 風險 | 對策 |
|------|------|
| 數學分數/根號是內嵌方程式物件,純文字抽出會缺字 | 轉換時偵測方程式物件 → 該題標記 `needs_review`,改抽成圖片或人工補 |
| 題目附圖(表格、幾何圖)需另存 | Word COM 匯出圖片,自動對應題號命名 |
| 2,240 檔逐一開啟耗時(估 1~3 小時) | 分科批次背景執行,逐科產出報告 |
| 國文題組(閱讀文章+多題)結構不同 | 題組另寫解析規則 |

## 4. 缺少的內容

- **英語聽力**:題庫無音檔,會考英聽需另外補(或第一版先不做聽力)
- 會考真題年度涵蓋 103~114,夠用 ✅
- 各單元易中難題量充足(單一單元抽樣即有 20+ 題)✅
