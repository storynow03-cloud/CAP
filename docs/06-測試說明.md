## 測試帳號(已建好,免信箱驗證)

|------|-------|------|---------|

⚠️ 注意:Supabase 預設新註冊帳號要收信點驗證連結才能登入。
若想跳過,到 Supabase 後台 → Authentication → Sign In / Providers → Email → 關閉「Confirm email」。

## 建議測試流程

1. **登入** → 應自動進入首頁儀表板(會考倒數、今日目標)
2. **自由練習** → 選數學 → 選一個單元 → 5 題 → 作答看詳解
3. **分階挑戰** → 選數學 → 連續答對幾題,觀察難度星數變化
4. **故意答錯 2 題** → 到「錯題本」確認它們出現(明天到期)
5. **學習歷程** → 確認雷達圖與每日統計有數字
6. 重新整理頁面/關瀏覽器再登入 → 紀錄應該都還在
