# 02 — 題庫格式規範(放題庫前必讀)

## 1. 資料夾結構

```
data/
└── questions/
    ├── chinese/      國文
    ├── english/      英語(含聽力題,音檔放 public/audio/)
    ├── math/         數學(含非選擇題)
    ├── science/      自然
    ├── social/       社會
    └── images/       題目附圖(幾何圖、圖表、地圖…)
```

每科一個或多個 JSON 檔,例如 `math/114年會考.json`、`math/一元二次方程式.json`。

## 2. 題目格式(JSON)

```json
{
  "id": "math-114-05",
  "subject": "math",
  "topic": "一元二次方程式",
  "subtopic": "配方法",
  "difficulty": 3,
  "type": "single_choice",
  "question": "已知 x² + 6x + a 可配方成 (x+3)² 的形式,求 a = ?",
  "image": null,
  "options": ["6", "9", "12", "36"],
  "answer": 1,
  "explanation": "配方:(x+3)² = x² + 6x + 9,所以 a = 9。",
  "source": "114年會考第5題",
  "tags": ["會考真題"]
}
```

## 3. 欄位說明

| 欄位 | 必填 | 說明 |
|------|:--:|------|
| `id` | ✅ | 全題庫唯一,建議格式 `科目-來源-題號` |
| `subject` | ✅ | `chinese` / `english` / `math` / `science` / `social` |
| `topic` | ✅ | 單元名稱(弱點分析、分階出題都靠它,**請務必填**) |
| `subtopic` | — | 更細的子主題 |
| `difficulty` | ✅ | 1~5(1 最簡單;對應會考:1-2 基礎、3 中等、4-5 精熟/挑戰) |
| `type` | ✅ | `single_choice` 單選 / `reading_group` 題組 / `listening` 聽力 / `non_choice` 數學非選 |
| `question` | ✅ | 題幹;數學式可用 LaTeX 寫法 `$x^2$`(系統會渲染) |
| `image` | — | 附圖檔名,放 `data/questions/images/` |
| `options` | 單選必填 | 選項陣列(不含 A/B/C/D 字母) |
| `answer` | ✅ | 正確選項索引(0 = A, 1 = B…);非選題填參考答案文字 |
| `explanation` | 建議 | 詳解(沒有詳解學生錯了學不到東西,強烈建議填) |
| `source` | — | 來源,如「114年會考」「自編」 |
| `tags` | — | 任意標籤 |

### 題組(閱讀測驗)格式
```json
{
  "id": "chinese-114-g1",
  "type": "reading_group",
  "passage": "（閱讀文章全文…）",
  "questions": [ { "id": "chinese-114-g1-1", "question": "…", "options": [...], "answer": 2, "...": "..." } ]
}
```

### 聽力題
`type: "listening"`,多一個 `"audio": "檔名.mp3"` 欄位,音檔放 `public/audio/`。

## 4. 你可以放的原始格式(我來轉換)

不一定要自己整理成 JSON。以下格式直接丟進 `data/raw/` 我就能轉:

- **Word(.doc/.docx)**:像「大姐的考卷」那種題目卷+解析卷
- **PDF**:歷屆會考試題卷
- **Excel/CSV**:一列一題
- **純文字/Markdown**

⚠️ 轉換時我會需要人工標 `topic`(單元)與 `difficulty`(難度),我會依題目內容判斷後給你過目確認。

## 5. 題庫內容建議清單(目前缺少、建議補齊)

| 優先序 | 內容 | 說明 |
|:--:|------|------|
| ★★★ | 歷屆會考真題(103~114年,五科) | 最有鑑別度,難度有官方依據 |
| ★★★ | 各科各單元的基礎題 | 分階挑戰的第 1~2 階需要大量簡單題 |
| ★★ | 各單元中等題、挑戰題 | 第 3~5 階 |
| ★★ | 英語聽力音檔 | 會考英聽佔成績等級 |
| ★ | 數學非選擇題 | 會考數學有 2 題非選 |
| ★ | 題目附圖 | 數學幾何、自然圖表、社會地圖很多題沒圖就不能用 |

**建議起步量**:每科每單元至少「易 5 題、中 5 題、難 3 題」,系統才跑得出分階效果。
