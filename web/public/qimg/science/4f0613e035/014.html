怡惠在學校的摸彩活動中抽到一個保溫杯，其構造如附圖。請回答下列問題：
(　　) (1)保溫杯的塑膠杯蓋，最主要是防堵熱的哪一個傳播途徑？　(A)傳導　(B)對流　(C)輻射　(D)杯蓋只是好看而已，無實際防堵熱量散失的功能。
(　　) (2)兩夾層中間抽成真空，可以防止熱的何種傳播途徑？　(A)傳導和對流　(B)對流和輻射　(C)傳導和輻射　(D)傳導、對流和輻射。
(　　) (3)夾層的表面鍍銀，主要可以防止熱的何種傳播途徑？　(A)傳導　(B)對流　(C)輻射　(D)三者皆可。

《答案》(1)A　(2)A　(3)C

詳解：(1)「塑膠杯蓋」是不良熱導體；(2)「傳導和對流」都需要介質，故抽成真空可以防止熱的傳導和對流；(3)表面鍍銀，可以防止熱的輻射。
